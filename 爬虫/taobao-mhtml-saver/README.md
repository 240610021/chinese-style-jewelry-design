# 淘宝店铺MHTML保存器 - QQ浏览器插件

## 安装步骤

1. 打开QQ浏览器
2. 在地址栏输入：`qqbrowser://extensions`
3. 打开"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本文件夹：`g:\Downloads\蔡规划\爬虫\taobao-mhtml-saver`

## 使用方法

1. **打开淘宝店铺页面**
   - 访问：`https://coppertistwu.jiyoujia.com/search.htm`
   - 确保已登录淘宝

2. **点击插件图标**
   - 在浏览器右上角找到插件图标（橙色"淘"字）
   - 点击打开弹窗

3. **开始保存**
   - 点击"开始保存商品"按钮
   - 插件会自动：
     - 提取所有商品链接
     - 逐个打开商品详情页
     - 滚动页面（模拟人类浏览）
     - 保存为MHTML格式

4. **查看进度**
   - 弹窗显示实时进度
   - 可以设置保存间隔和最大商品数
   - 随时可以点击"停止保存"

5. **导出商品列表**
   - 点击"导出商品列表"可导出JSON格式的商品信息

## 文件说明

- `manifest.json` - 插件配置文件
- `popup.html` - 弹窗界面
- `popup.js` - 弹窗逻辑
- `background.js` - 后台服务
- `content.js` - 页面注入脚本

## 保存位置

MHTML文件会自动下载到浏览器的默认下载文件夹，文件名格式：
```
{商品标题}_{商品ID}-淘宝网.mhtml
```

## 注意事项

1. 确保在店铺搜索页使用（URL包含 `search.htm`）
2. 保存过程中不要关闭浏览器
3. 首次使用需要授权调试权限
