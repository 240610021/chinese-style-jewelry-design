# 入职90天行动方案

> 假设岗位：**国潮银饰品牌经理 / 产品运营负责人**
> 目标：用90天完成从调研诊断到策略输出再到落地验证的完整闭环，证明"既能看大局，也能打胜仗"。

---

## 一、整体框架

```
第1月（Day 0-30）：全面调研诊断 —— "知道我们在哪"
第2月（Day 31-60）：策略方案输出 —— "知道要去哪"
第3月（Day 61-90）：关键战役落地 —— "开始走过去"
```

---

## 二、第1月：调研诊断（Day 0-30）

### 2.1 目标
- 摸清公司现有产品、供应链、渠道、团队的基线水平
- 建立行业竞争坐标系，找到差异化机会点
- 识别最短板的3个关键问题

### 2.2 具体动作

| 周次 | 动作 | 交付物 |
|------|------|--------|
| **W1** | 产品审计：盘点现有全部SKU，按"产品潜力评分模型"打分 | 《产品竞争力评估表》 |
| **W1** | 供应链摸底：拜访工厂，记录起订量、打样周期、良品率 | 《供应链基线报告》 |
| **W2** | 竞品监控：锁定3-5家核心竞品，监控其上新节奏、价格、销量、评价 | 《竞品动态监测周报》 |
| **W2** | 用户洞察：翻阅近3个月客服聊天记录、退货原因、差评内容 | 《用户痛点清单Top20》 |
| **W3** | 渠道诊断：分析各渠道（天猫/抖音/小红书）的ROI、转化率、客单价 | 《渠道效率诊断报告》 |
| **W3** | 团队访谈：与设计师、运营、客服1对1沟通，了解内部卡点 | 《内部协作瓶颈清单》 |
| **W4** | 数据整合：输出完整的《品牌现状诊断报告》 | 《现状诊断报告》+ 《机会点清单》 |

### 2.3 关键交付物：《品牌现状诊断报告》框架

1. **数据概览**：现有GMV、毛利率、复购率、客单价
2. **产品矩阵图**：按价格带×文化主题画矩阵，标注竞争力强弱
3. **竞品对标表**：与琢匠、太禾天喜等头部在5个维度打分对比
4. **用户痛点Top10**：按提及频次排序
5. **机会点清单**：3个短期机会（3个月内可做）+ 2个中期机会（6个月内可做）

## 三、第2月：策略输出（Day 31-60）

### 3.1 目标
- 基于第1月诊断，输出可落地的年度策略
- 确定Q1的产品、渠道、内容三大战役
- 建立数据监测体系，为后续优化埋点

### 3.2 具体动作

| 周次 | 动作 | 交付物 |
|------|------|--------|
| **W5** | 产品策略：确定Q1上新计划，选出1款S级旗舰+2款A级主力 | 《Q1产品企划书》 |
| **W5** | 定价策略：基于银价-成本-定价联动模型，确定新价格体系 | 《定价策略手册》 |
| **W6** | 渠道策略：重新分配各渠道预算，确定主攻渠道 | 《渠道预算分配表》 |
| **W6** | 内容策略：制定Q1内容日历，确定3个核心传播话题 | 《Q1内容营销日历》 |
| **W7** | 私域策略：设计会员体系+裂变机制 | 《私域运营SOP》 |
| **W7** | 监测体系：搭建核心数据看板（日/周/月维度） | 《数据监测看板V1.0》 |
| **W8** | 方案整合：召开策略评审会，输出《年度经营计划》 | 《年度经营计划》 |

### 3.3 关键交付物：《年度经营计划》核心模块

```
一、年度目标
   - GMV目标：XXX万（分解到月/渠道/系列）
   - 毛利率目标：XX%
   - 复购率目标：XX%

二、产品策略
   - 系列矩阵：云纹/山海/乾坤各自上新节奏
   - 爆款策略：S级产品定义及资源倾斜方案
   - 淘汰机制：月销量低于X件的SKU自动进入清仓

三、渠道策略
   - 天猫：搜索优化+直通车预算分配
   - 抖音：达人合作名单+自播排期
   - 小红书：KOC矩阵+关键词布局

四、内容策略
   - 月度话题：每月1个核心传播主题
   - 内容模板：主图文案模板、短视频脚本模板
   - 达人合作：腰尾部KOC为主，头部KOL为辅

五、财务预算
   - 营销费用：XX万（占GMV XX%）
   - 产品研发：XX万
   - 人员成本：XX万
```

## 四、第3月：落地验证（Day 61-90）

### 4.1 目标
- 推动至少1款新产品完成从设计到上架的全流程
- 启动至少1场有数据产出的营销战役
- 建立可复用的工作SOP，为后续规模化打基础

### 4.2 具体动作

| 周次 | 动作 | 交付物 |
|------|------|--------|
| **W9** | 产品落地：跟进S级产品打样、拍摄、详情页制作 | 产品上架 |
| **W9** | 内容落地：产出首批小红书笔记/短视频素材 | 10篇笔记 + 3条视频 |
| **W10** | 战役启动：S级产品上新 campaign 全渠道启动 |  Campaign 执行 |
| **W10** | 投放测试：天猫直通车/抖音千川小规模测试 | A/B测试数据 |
| **W11** | 数据复盘：Campaign 第1周数据复盘，快速调整 | 《战役中期复盘报告》 |
| **W11** | 私域启动：首批会员招募+社群搭建 | 500人社群 |
| **W12** | 90天总结：输出《90天工作总结》，包含成果、教训、下一步 | 《90天工作总结》 |

### 4.3 关键交付物：《90天工作总结》框架

1. **量化成果**
   - 产品：X款新品上架，S级产品首周销量XXX件
   - 流量：新增访客XX万，转化率X%
   - 内容：产出XX篇内容，总曝光XX万
   - 私域：沉淀用户XXX人

2. **关键发现**
   - 用户最买单的3个卖点
   - 转化率最高的1个渠道/素材
   - 成本最低的1个获客路径

3. **踩过的坑**
   - 至少3个真实教训（展示反思能力）

4. **下一步计划**
   - Q2的3个核心目标
   - 需要的资源支持

## 五、关键能力展示

### 5.1 这份方案展示了什么

| 能力 | 体现 |
|------|------|
| **行业认知** | 用一手调研数据（173款样本分析）支撑诊断，而非拍脑袋 |
| **系统思维** | 从调研→策略→落地的完整闭环，而非零散点子 |
| **数据驱动** | 每个阶段都有量化目标和数据监测看板 |
| **落地能力** | 具体到每周做什么、交付什么，可直接执行 |
| **风险意识** | 包含踩坑复盘和资源需求，不盲目承诺 |

### 5.2 适配不同岗位的侧重点

| 应聘岗位 | 侧重点调整 |
|----------|-----------|
| **品牌经理** | 强化品牌定位、视觉体系、内容策略模块 |
| **产品经理** | 强化产品审计、评分模型、供应链对接模块 |
| **电商运营** | 强化渠道诊断、投放测试、数据复盘模块 |
| **内容运营** | 强化用户洞察、内容日历、KOC矩阵模块 |

---

> **核心信息**：我不需要90天来"熟悉环境"，我可以在入职第一天就带着这套框架开始干活。这份方案证明的是——**我不仅懂这个行业，我还知道怎么赢。**
