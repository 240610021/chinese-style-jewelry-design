# 东南亚国潮银饰出海执行方案

> 基于琢匠等头部品牌已验证的出海路径，输出以 TikTok Shop + Shopee 为主阵地的东南亚启动方案。证明"国内市场我能打，海外增量我也看得懂"。

---

## 一、为什么是现在？

### 1.1 东南亚市场机会

| 指标 | 数据 | 信号 |
|------|------|------|
| 东南亚电商增速 | 2025年+20%（全球最高） | 流量红利期 |
| TikTok Shop 东南亚GMV | 2025年超300亿美元 | 内容电商爆发 |
| 华人人口 | 东南亚约3000万 | 文化认同基础 |
| 国潮认知度 | 新加坡/马来西亚Z世代对中国文化好感度>60% | 品牌接受度高 |
| 竞品密度 | 国潮银饰卖家极少， mostly 韩国/日本饰品 | 蓝海 |

### 1.2 东南亚 vs 国内 核心差异

| 维度 | 国内 | 东南亚 |
|------|------|--------|
| **主力价格带** | 200-800元 | 80-300元（折合人民币） |
| **决策驱动** | 文化认同+社交货币 | 性价比+网红推荐 |
| **支付方式** | 支付宝/微信 | COD（货到付款）占比40%+ |
| **物流时效** | 1-3天 | 5-15天（跨境） |
| **宗教敏感** | 佛教/道教文化可用 | 马来西亚/印尼需避开伊斯兰教禁忌 |

## 二、双平台启动策略

### 2.1 TikTok Shop：内容获客主阵地

**定位**：用短视频+直播打爆品，建立品牌认知

| 模块 | 策略 |
|------|------|
| **账号矩阵** | 1个品牌官方号（英文/中文双语）+ 3-5个本地达人分销号 |
| **内容策略** | ① 工艺类：錾刻过程（无国界，视觉冲击强）② 穿搭类：东南亚风格搭配 ③ 文化类：用英文讲中国符号故事 |
| **达人合作** | 首批50个腰尾部达人（1-10万粉），寄样+佣金模式（15-25%） |
| **直播策略** | 每周3场，每场2小时，主打"限时闪购"（东南亚用户对限时折扣敏感） |
| **定价策略** | 国内468元款 → 东南亚定价89-129新币/马币（约480-700人民币，含跨境溢价） |

### 2.2 Shopee：搜索电商基本盘

**定位**：承接精准搜索流量，做日常销量

| 模块 | 策略 |
|------|------|
| **店铺定位** | 旗舰店形象，全系列上架，主打"Chinese Cultural Jewelry" |
| **关键词布局** | 英文：dragon bracelet / chinese knot / copper silver jewelry；马来文/泰文：用本地语言覆盖 |
| **主图策略** | 白底产品图+文化符号标注（东南亚用户喜欢"看得懂"的设计） |
| **促销活动** | 每月参加Shopee官方大促（9.9、11.11、12.12），设置满减+包邮 |
| **评价运营** | 前100单主动邀评，送小礼品（如中国结挂饰）换五星 |

## 三、产品本地化调整

### 3.1 需调整的要素

| 要素 | 国内版 | 东南亚版 | 原因 |
|------|--------|----------|------|
| **尺寸** | 按中国手围 | 加大0.5-1cm | 东南亚平均体型偏大 |
| **包装** | 古典中国风 | 简约国际化+中文元素点缀 | 降低文化隔阂 |
| **说明书** | 中文 | 英文+当地语言 | 合规+用户体验 |
| **寓意强调** | 龙、饕餮、太极 | 龙（通用）、锦鲤（好运）、莲花（佛教友好） | 避开宗教禁忌 |
| **价格** | 468元 | 89新币/299马币 | 符合当地购买力 |

### 3.2 宗教禁忌清单

| 国家 | 禁忌 | 替代方案 |
|------|------|----------|
| **马来西亚/印尼** | 避免猪、狗形象 | 用龙、鱼、莲花 |
| **泰国** | 避免脚朝向佛像的设计 | 产品设计不强调方向性 |
| **菲律宾** | 天主教为主，相对开放 | 可用十字架+中国结融合设计 |
| **新加坡** | 多元文化，接受度高 | 全系列可上，但需标注文化出处 |

## 四、物流与供应链方案

### 4.1 物流模式选择

| 模式 | 时效 | 成本 | 适合阶段 |
|------|------|------|----------|
| **国内直发** | 7-15天 | 低 | 启动期（月销<500单） |
| **第三方海外仓** | 2-3天 | 中 | 增长期（月销500-2000单） |
| **自建本地仓** | 1-2天 | 高 | 稳定期（月销>2000单） |

### 4.2 启动期物流方案（国内直发）

- **发货地**：深圳/广州（靠近水贝供应链）
- **物流商**：云途/燕文/4PX（专线小包，时效7-12天）
- **运费**：约15-25元/件（50g以内），建议设置满XX包邮门槛
- **COD处理**：与物流商合作代收货款，手续费3-5%

### 4.3 关税与合规

| 国家 | 免税额度 | 关税税率 | 备注 |
|------|----------|----------|------|
| 新加坡 | 400新币以下 | 0% | 最受欢迎市场 |
| 马来西亚 | 500马币以下 | 0-10% | 需注册SST |
| 泰国 | 1500泰铢以下 | 0% | 超过需缴VAT 7% |
| 印尼 | 3美元以下 | 0% | 清关复杂，建议暂缓 |
| 菲律宾 | 10000比索以下 | 0% | 市场潜力大 |

## 五、首年里程碑与预算

### 5.1 里程碑

| 时间 | 目标 |
|------|------|
| **第1-2月** | 完成TikTok Shop/Shopee店铺注册、产品上架、首批50个达人寄样 |
| **第3-4月** | 首月GMV破1万人民币，积累100条用户评价 |
| **第5-6月** | 跑通1个爆款（月销>200件），ROI>1.5 |
| **第7-9月** | 拓展至2个国家（如新加坡+马来西亚），月GMV破5万 |
| **第10-12月** | 启动海外仓测试，年GMV目标20-30万人民币 |

| 物流运费 | 20,000 | 20% |
| 翻译+本地化 | 10,000 | 10% |
