# 供应链落地清单

> 基于错铜工艺的生产特性，输出从原材料采购到成品入库的全链路供应商类型、起订量、周期及成本参考。证明"策略不是纸上谈兵，我知道货从哪来"。

---

## 一、错铜工艺供应链全景图

```
原材料层
├── 银料供应商（上海黄金交易所/水贝批发商）
├── 铜料供应商（江西铜业/云南铜业经销商）
└── 辅助材料（錾子、磨料、抛光膏）

制造层
├── 錾刻工坊（核心工艺，人工密集型）
├── 铸造厂（铜基底铸造/银板压延）
├── 表面处理厂（抛光、做旧、防氧化）
└── 组装包装厂（串珠、穿绳、礼盒包装）

品控层
├── 第三方检测机构（贵金属含量检测）
└── 内部质检（外观、尺寸、重量）
```

## 二、核心供应商类型与对接要点

### 2.1 银料采购

| 项目 | 详情 |
|------|------|
| **采购渠道** | 上海黄金交易所（正规、价格高）、深圳水贝批发商（灵活、价格优） |
| **纯度要求** | S925（925银），需提供材质证明 |
| **采购形态** | 银板/银条（压延用）、银珠/银丝（镶嵌用） |
| **价格参考** | 按当日银价+加工费（约0.5-1元/克） |
| **起订量** | 水贝批发商通常500g起批 |
| **账期** | 现款现货为主，长期合作可谈月结 |

### 2.2 铜料采购

| 项目 | 详情 |
|------|------|
| **采购渠道** | 江西铜业/云南铜业经销商、本地金属材料市场 |
| **材质要求** | 紫铜（T2纯铜），延展性好，适合錾刻 |
| **采购形态** | 铜板/铜丝（錾刻用）、铜珠（镶嵌用） |
| **价格参考** | 约60-80元/公斤，价格波动小 |
| **起订量** | 10公斤起批 |

### 2.3 錾刻工坊（核心环节）

| 项目 | 详情 |
|------|------|
| **分布地区** | 云南大理/丽江（白族银匠传统）、贵州黔东南（苗族银饰）、福建莆田（珠宝加工集群） |
| **合作模式** | ① 包工包料（省心，利润薄） ② 来料加工（控成本，需自己管材料） |
| **起订量** | 单款30-50件起做（太少师傅不接） |
| **打样周期** | 7-15天（复杂款20天+） |
| **大货周期** | 单款100件约20-30天 |
| **工费参考** | 简单款15-30元/件，复杂款50-100元/件 |
| **关键考察点** | ① 师傅工龄（10年+为佳） ② 成品一致性 ③ 能否看图打样 |

### 2.4 铸造/压延厂

| 项目 | 详情 |
|------|------|
| **功能** | 铜基底铸造、银板压延成薄片 |
| **分布地区** | 深圳水贝、福建莆田、广州番禺 |
| **起订量** | 铜模开模费500-2000元/款，批量100件起 |
| **周期** | 开模3-5天，批量7-10天 |

### 2.5 表面处理厂

| 项目 | 详情 |
|------|------|
| **功能** | 抛光、做旧（硫化处理）、防氧化涂层 |
| **关键工艺** | 做旧：硫化钾溶液浸泡，形成古朴铜绿；防氧化：纳米涂层或透明漆 |
| **周期** | 批量3-5天 |
| **成本** | 2-5元/件 |

### 2.6 检测机构

| 项目 | 详情 |
|------|------|
| **推荐机构** | 国家珠宝玉石质量监督检验中心（NGTC）、各地贵金属检测站 |
| **检测项目** | 银含量、铜含量、镍释放量（防过敏） |
| **周期** | 3-5个工作日 |
| **费用** | 100-300元/批次 |
| **用途** | 入驻天猫/京东的资质文件、消费者信任背书 |

## 三、首年供应链配置建议

### 3.1 阶段一：启动期（0-3个月）

| 配置 | 策略 |
|------|------|
| **錾刻工坊** | 绑定1家主供应商（云南或贵州），签框架协议，锁定产能 |
| **起订量** | 每款打样1件 → 试单30件 → 根据销量补单 |
| **库存策略** | 少备现货，以预售/定制为主，降低库存风险 |
| **检测** | 首批送检，获取NGTC证书，后续按批次抽检 |

### 3.2 阶段二：增长期（4-9个月）

| 配置 | 策略 |
|------|------|
| **錾刻工坊** | 拓展至2-3家，分散产能风险 |
| **起订量** | 爆款单款100-200件，平款50件 |
| **库存策略** | 爆款备1.5个月库存，平款备0.5个月 |
| **标准化** | 输出《工艺标准手册》，统一各工坊的錾刻深度、抛光亮度 |

### 3.3 阶段三：稳定期（10-12个月）

| 配置 | 策略 |
|------|------|
| **錾刻工坊** | 3家稳定合作+1家备用 |
| **起订量** | 爆款300件+，利用规模压降工费 |
| **自建考虑** | 若年销破千万，可考虑自建小型錾刻车间（5-8人） |

## 五、供应商考察Checklist

首次拜访錾刻工坊时，带上这份清单：

- [ ] 查看师傅现场錾刻，观察手法是否熟练
- [ ] 要求提供3件同款式样品，对比一致性
- [ ] 询问工龄、师承、日均产量
- [ ] 查看工作环境的整洁度（影响产品清洁度）
- [ ] 确认能否签保密协议（防止款式外泄）
- [ ] 确认交付周期及逾期赔偿条款
- [ ] 索要过往合作品牌案例（如有）
- [ ] 测试沟通效率：发一张设计图，看多久能给出报价和周期
