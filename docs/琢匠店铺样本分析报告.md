## 五、店铺整体判定

### 1. 生意健康度：★★★★☆
- 40款样本总销量超3万件，平均750件/款，动销率极高
- 没有零销量商品（样本内），说明供应链和流量运营成熟

### 2. 品类策略："配件养店，摆件立牌"
- **手串配件**占SKU一半，承担流量和复购功能，客单价低但周转快
- **金属摆件/手办**占37.5%，是品牌差异化核心，客单价高、利润厚
- **吊坠/挂件**占比10%，是潜在的第二增长曲线

### 3. 设计风格：三条主线并行
| 主线 | 代表商品 | 目标人群 |
|------|----------|----------|
| **道系/藏传** | 山鬼、三清铃、九宫八卦、金刚杵、尸陀林 | 文玩硬核玩家 |
| **三国/历史** | 关羽、张飞、战马、翁仲 | 泛国潮男性消费者 |
| **机械/现代** | 机甲裁决者、机械心脏、黑骑士、圣骑士 | 潮玩/Z世代 |

### 4. 风险点
- **有龙则灵**（龙头三通）销量仅15件，与品牌同系列其他产品落差巨大，可能存在：
  - 定价过高
  - 主图/详情转化差
  - 上架时间短
- **定制类产品**（翁仲全银定制）销量天然受限，需控制库存和交付周期

### 5. 对竞品的启示
- 琢匠的成功公式 = **S925银/黄铜材质** + **强文化符号**（道教、佛教、三国） + **可玩性设计**（磁吸、可动、拼装）
- 手串配件是"钩子产品"，用来降低新客试错成本；摆件是"利润产品"，用来拉升客单价
- 如果与鸡鸣寺合作，可借鉴其"小配件引流 + 大摆件/佛像立品牌"的SKU结构
