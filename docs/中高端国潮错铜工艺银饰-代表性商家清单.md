# 中高端国潮银饰（错铜工艺）代表性商家清单

> 本文件聚焦项目核心方向——中高端国潮银饰，尤其以**错铜工艺**（银铜错/金银错）为核心工艺特色的代表性商家，供竞争对标与差异化定位参考。

---

## 一、错铜工艺说明

错铜工艺（又称银铜错、金银错）是中国传统金属细工技艺，源于春秋战国时期的"金银错"。其核心是在银器表面錾刻出图案凹槽，再嵌入铜丝或铜片，经打磨抛光后形成银底铜纹的对比效果。该工艺兼具：
- **视觉张力**：银白与铜红的冷暖对比，复古质感强烈
- **文化厚度**：非遗技艺背书，国潮叙事天然载体
- **价值溢价**：纯手工制作，每件独一无二，支撑中高端定价

> **风格细分说明**：本清单中部分品牌虽非严格以"错铜"为主工艺，但因其采用**银铜结合/做旧/多材质镶嵌**等相近工艺语言，且同属中高端国潮赛道，故一并纳入对标参考。其中：
> - **次世代/暗黑国潮风格**：无垠（混沌巨蛇IP）、巫羿（骷髅/神话图腾）—— 风格更偏暗黑、二次元、亚文化，价格带更高，社群粘性极强
> - **传统国潮风格**：琢匠、太禾天喜、百年宝诚等 —— 风格更偏传统国风、道系/神话IP

## 四、本项目差异化机会点

基于以上商家分析，中高端国潮错铜工艺银饰的**差异化切口**建议：

1. **工艺极致化**：在错铜基础上叠加花丝/珐琅/做旧，形成"复合工艺"壁垒
2. **场景细分化**：避开手镯红海，切入男士银饰、情侣对饰、宠物铭牌等蓝海
3. **内容IP化**：以"错铜工艺纪录片+匠人直播"建立信任，对标九乐抖音模式
4. **定制深度化**：提供"图案设计→铜丝配色→刻字"全流程C2M定制
5. **出海前置化**：借鉴琢匠COPPERTIST WU路径，以东方美学切入东南亚/欧美市场
